# Algorand Project

## 💡 Project Idea
Peer-to-Peer Payment System using Algorand.

## 📌 Description
This project allows users to send and receive ALGO securely without intermediaries.

## 🚀 Features
- Fast transactions
- Low fees (0.001 ALGO)
- Secure and decentralized
